# ProyectoColaborativo - Sistema de transporte
## Descripción
Este proyecto permite la aplicación de los principios de la POO (herencia, abstracción, polimorfismo, etc).
El mismo simula una red de transporte en donde el sistema permite la interacción entre dos tipos de vehículos y
un núcleo de control.
