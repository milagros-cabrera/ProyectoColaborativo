# ProyectoColaborativo - Sistema de transporte
## Descripción
Este proyecto permite la aplicación de los principios de la POO (herencia, abstracción, polimorfismo, etc).
El mismo simula de manera sencilla una red de transporte en donde el sistema permite la interacción entre dos tipos de vehículos y
un núcleo de control.
* **Integrante A** [@milagros-cabrera](https://github.com/milagros-cabrera)
* **Integrante B** [@ceciliaromero-creator](https://github.com/ceciliaromero-creator)
## Instrucciones de uso: 
1. Crear una clase *Main*.
2. Instanciar el centro de control (*CentroControl*).
3. Instanciar objetos de *CamionDeCarga* y *TransportePublico*.
4. Llamar al método **regirtrarUnidad()** pasando los objetos creados como argumentos.
5. Llamar al método **monitorearFlota()**.
6. Llamar directamente al método **sincronizarGPS()** desde la instancia *CamionDeCarga*.
